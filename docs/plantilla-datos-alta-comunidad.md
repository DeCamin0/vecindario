## Variantes de envío (resumen para el equipo)

1. **Correo + este documento** — Adjuntas el PDF/Word generado desde este `.md` o pegas las tablas en el cuerpo del mail.  
2. **Google/Microsoft Forms** — Mismas preguntas como campos de formulario; respuestas en hoja de cálculo.  
3. **PDF rellenable** — Exportar a PDF con campos de formulario (Acrobat, Canva, etc.).  
4. **Notion / enlace único** — Duplicas la estructura en una página con base de datos.

Si falta algún dato, lo solicitamos por correo; los campos técnicos avanzados los ajustamos nosotros en back-office.
