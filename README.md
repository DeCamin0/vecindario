# Vecindario

Aplicație comunități: web (Vite/React), API (Node/Express), Prisma/MySQL.

- **Web:** `npm install` → `npm run dev` (Vite, port implicit 5175).
- **API:** `npm install --prefix server` → vezi `docs/DEPLOY-VPS.md`.
- **Variabile:** copiază `.env.example` → `.env` în rădăcina proiectului (nu se comite).

Documentație deploy: [docs/DEPLOY-VPS.md](docs/DEPLOY-VPS.md).
